### 迭代 4

命令：

```bash
python3 -m json.tool plugins/signalproof/.codex-plugin/plugin.json
python3 -m json.tool .agents/plugins/marketplace.json
python3 scripts/signalproof.py check-goal --min-cases 5
```

结果：

```text
SignalProof goal check: PASSED
```

发现：

repo marketplace 路径必须相对 `.agents/plugins/marketplace.json` 解析，而不是相对仓库根目录。

优化：

把 `.agents/plugins/marketplace.json` 改成 `../../plugins/signalproof`，并在 `check-goal` 中加入 plugin/marketplace 路径解析检查。
