# SignalProof HyperFrames 视觉规范

## Style Prompt

克制、技术向、证据优先。画面像一份会动的研究摘要，不做营销炫技；以清晰信息层级、稳定网格和少量高对比强调传达“插件真实试跑”的结论。

## Colors

- 背景：`#F7F8FA`
- 主文字：`#111827`
- 辅助文字：`#4B5563`
- 强调青绿：`#0F766E`
- 风险琥珀：`#D97706`
- 线框：`#D7DEE2`

## Typography

- 主字体：system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- 标题：粗体，大字号，短句
- 正文：中等字号，控制行长

## What NOT to Do

- 不使用紫蓝渐变、光球、装饰性背景。
- 不把插件名字堆成密集小字。
- 不做无法解释的数据动画。
- 不使用随机动画或无限循环。
