# Feedback Form Fields

- Tool: Codex / Claude Code / Cursor / Other.
- Repo type: frontend / backend / full-stack / CLI / other.
- Can the task be public: public / anonymized / private.
- Original task description.
- What did the AI get wrong?
- Do you currently have `AGENTS.md` / `CLAUDE.md`?
- Biggest context problem: too long / too generic / no verification command / no business contract / no directory boundary / other.
- What do you want improved?
- Are you willing to review a before/after result?
- Are you willing to publish an anonymized case?
- Contact.

## Feedback Quality

- A: real task + concrete failure + willing to review before/after.
- B: concrete objection or improvement suggestion.
- C: like, star, bookmark, generic support.
- D: unrelated discussion.

Only A/B count as validation signals.
