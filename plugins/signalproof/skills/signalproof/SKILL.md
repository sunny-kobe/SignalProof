# SignalProof Plugin Skill

SignalProof 是一套 Codex 优先的工作流协议。

调用时，创建或更新一个本地 case，并覆盖：

- 信号捕捉；
- 研究；
- 反方审查；
- 判断；
- 验证计划；
- 产物；
- 反馈边界；
- 决策；
- 资产；
- 流程自检；
- 工具账本；
- 过程日志。

除非存在真实用户反馈，否则不要声称市场验证。
